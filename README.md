# 純前端背單字 App

這是一個純前端、無框架的單字背誦應用程式，包含：

- `index.html`：單字卡與管理頁面
- `styles.css`：介面樣式
- `script.js`：功能邏輯

## 功能

- 中央卡片點擊可翻頁
- 正面顯示英文單字
- 背面顯示翻譯、詞性、例句、字根分析
- 管理頁面可新增/刪除單字
- 管理頁面有「自動填入」按鈕，會呼叫公開 API 自動查詢翻譯與詞性、例句、字根分析
- 單字資料儲存在瀏覽器 `localStorage`

## 使用方式

直接打開 `index.html` 即可使用。若想重新開始，可清除瀏覽器 localStorage。
